Basic Interactive Elements

In-text Link

Using in-text link to transport users to another page or another sections.

Example

Here is the link within a line of text will link you to the page of #.

Code Snippet

<p>Here is the <a class="IntextLink" href="#" abbr title="Go to the page of #">link</a>within a line of text will link you to the page of #.</p>

Call To Action Button

Using CTA button to call users' action.

Example

START TO SHOP

Code Snippet

<p><a class="CallToActionButton" href="#" abbr title="Go to the page of #">START TO SHOP</a></p>

Submit and Reset Button

Using submit and reset button when users need to submit or reset information.

Example of Submit Button

Code Snippet of Submit Button

<button class="SubmitButton">Submit</button>

Example of Reset Button

Code Snippet of Reset Button

<button class="ResetButton">Reset</button>

Text Input Field and Label

Using text input field and label when users need to input or complete a form, such as sign up form.

Example

Code Snippet

<label class="SignInForm">Username*</label>					<input type="text" class="SignInForm" placeholder="">

Headings

Heading 1

Using heading1 for title of functional pages (eg:Account).

Code Snippet

<h1>Heading 1</h1>

Heading 2

Using heading2 for title of functional section (eg:Billing address).

Code Snippet

<h2>Heading 2</h2>

Heading 3

Using heading3 for title of inactivate tabs and lables of forms.

Code Snippet

<h3>Heading 3</h3>

Using heading4 for name of products on product list page/ Filter.

Code Snippet

<h4>Heading 4</h4>

Paragraph

Example

This is the example of paragraph. We can use strong to emphasize special point. Or we can use regular size for normal text. Such as: Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.

Code Snippet

							<p>This is the example of paragraph. <strong>We can use strong to emphasize special point.</strong> Or we can use regular size for normal text. Such as: Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.</p>

Bulleted List

Using bulleted list for benefits of products.

Example

  • Free of preservatives, additives or fillers
  • Simple, natural, whole ingredients
  • Balanced & complete according to AAFCO guidelines
  • Heart-healthy. Omega-3 Fish oil & Organics Safflower/Flaxseed Oil

Code Snippet

<ul class="ProductBenefit">
									<li>Free of preservatives, additives or fillers</li>
									<li>Simple, natural, whole ingredients</li>
									<li>Balanced & complete according to AAFCO guidelines</li>
									<li>Heart-healthy. Omega-3 Fish oil & Organics Safflower/Flaxseed Oil</li></ul>

Numbered List

Example

Code Snippet
